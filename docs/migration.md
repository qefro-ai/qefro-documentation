## REST-only to SDK tools
Migrate incrementally without breaking existing integrations.

### Phase 1
Keep REST and OpenAPI tools active.

### Phase 2
Move auth-sensitive actions to SDK handlers.

### Phase 3
Sync SDK tools and update prompts for mixed routing.

### Phase 4
Retire redundant REST actions after parity validation.
