# Website Widget

The **Website Widget** embeds Customer AI on your site.

## Introduction

Website Widget is a core part of the Qefro AI Workspace Platform. This page defines the concept, how it fits the architecture, and how to operate it safely.

## Why it exists

Organizations need clear boundaries for website widget so Customer AI and Employee AI stay accurate, permissioned, and auditable.

## Concepts

- Definition aligned with Qefro terminology
- Relationship to workspaces, knowledge, and Business Actions
- Operational settings in the Admin Console

## Architecture

embed→jwt→stream

```mermaid
sequenceDiagram
  participant Site
  participant Widget
  participant API
  Site->>Widget: Load script
  Widget->>API: Auth JWT
  Widget->>API: Chat WebSocket
```
