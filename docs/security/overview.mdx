# Security Overview

**Security Overview** — Tenant isolation, encrypted secrets, SSRF, identity forwarding, audit logs.

## Introduction

Qefro security controls in the running product:

- Multi-tenant isolation by `tenant_id`
- Workspace isolation for knowledge/tools/conversations
- Encrypted Business Tool secrets
- SSRF / DNS-pinned egress for tools and handoff webhooks
- End-user identity forwarding via `identify()`
- Org audit logs (`/api/v1/org/audit-logs`) and tool execution logs
- Auth abuse rate limits
- Razorpay webhook signature verification

SOC 2 and SSO/SAML are on the roadmap — contact Sales for timelines.

## Why it exists

Security documentation must reflect implemented controls, not aspirational marketing.
