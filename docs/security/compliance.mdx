# Compliance

**Compliance** — Roadmap items: SOC 2, SSO/SAML, Enterprise private deploy.

## Introduction

Current public posture: strong isolation + encryption + auditability on the cloud product. Formal SOC 2 certification and SSO/SAML are roadmap items — email support@qefro.com / Sales for timelines. Enterprise private deployment is available as a custom engagement.

## Why it exists

Security documentation must reflect implemented controls, not aspirational marketing.
