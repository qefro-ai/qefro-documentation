# Enable Custom Domains

**Enable Custom Domains** is a practical guide: Point DNS at your Internal Portal.

## Introduction

Follow this guide to point dns at your internal portal. using the Qefro Admin Console and documented APIs.

## Why it exists

Guides encode the recommended path so teams avoid insecure shortcuts and incomplete go-lives.

## Concepts

- Prerequisites (plan, roles, workspace)
- Configuration steps
- Validation and rollback

## Architecture

Guide steps map to Admin Console configuration and optional API calls.

```mermaid
flowchart TD
  A[Prerequisites] --> B[Configure]
  B --> C[Validate]
  C --> D[Go live]
  D --> E[Monitor]
```

## Best practices

- Keep a staging workspace for experiments
- Record owners and runbooks before production
- Re-test after every OpenAPI/tool change
