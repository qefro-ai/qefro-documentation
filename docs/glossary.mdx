# Glossary

| Term | Definition |
| --- | --- |
| Organization / Tenant | Top-level account boundary for billing and isolation |
| AI Workspace | Isolated knowledge + tools + conversations |
| Customer AI | Website widget and/or WhatsApp assistants |
| Employee AI | Internal Portal assistants for org members |
| Knowledge Platform | Ingest + hybrid retrieval + citations |
| Business Tool | REST/OpenAPI connector with encrypted credentials |
| Business Action | Runtime invocation of a Business Tool |
| Identity Forwarding | identify() end-user JWT/session into tool calls |
| Widget token | Publishable embed key for cdn.qefro.com/widget.js |
| Internal Portal | your-company.qefro.com employee experience |
| Owner / Admin / Member | Organization RBAC roles |
| Team | Grants Members access to workspaces |
